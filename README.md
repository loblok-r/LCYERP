### 2.预算数据导出深分页问题   方案：子查询分页，游标分页
#### 实验思路 ：模拟真实数据环境->复现深分页问题->测试接口->实现游标分页方案->测试接口->文档整理
### 3.大数据量资产明细数据导出为EXCEL表格
#### 实验思路

| 阶段                        | 目标                   | 技术手段                                    | 关键验证点                   |
| --------------------------- | ---------------------- | ------------------------------------------- | ---------------------------- |
| **1. 原始方案（Baseline）** | 复现“内存爆炸”问题     | `SELECT *` + `XSSFWorkbook` 全量加载        | OOM / 内存峰值 / 耗时        |
| **2. 子查询分页优化**       | 解决内存问题，稳定导出 | 游标 or 子查询分页 + `SXSSFWorkbook` 流式写 | 内存可控、无 OOM、耗时下降   |
| **3. 多线程并行导出**       | 在稳定基础上提速       | 分片 + `CompletableFuture` + 合并           | 耗时进一步下降、资源利用率合 |
### 4.企业多银行渠道薪资代发接口对接与路由调度
云ERP背景，不同客户使用不同银行（如工行、招行、建行）进行薪资发放，系统需根据企业配置自动选择对应银行通道，并处理异构接口差异。
**技术点覆盖**：
- 定时任务（`@Scheduled`）
- 策略模式 + 工厂
- 幂等性（虽本次无重复提交，但 biz_id 已存在）
- 批量处理 + 事务控制
- 消息队列
- 分布式锁
- 状态机
- 事件驱动
